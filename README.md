# BART Command-line tool

```sh
Usage: python bart.py [-h] [-n] [-s] [station]

  station     Station name

optional arguments:
  -h, --help  show this help message and exit
  -n          Show northbound
  -s          Show southbound
```

e.g.

```sh
$ ./bart.py POWL -n
```

Default station is Powell, default direction is southbound trains.
